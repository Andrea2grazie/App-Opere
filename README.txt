ISTRUZIONI RAPIDE

1) Vai su Supabase e crea un progetto gratuito.
2) Apri SQL Editor e incolla il contenuto di schema_supabase.sql.
3) Vai in Project Settings > API.
4) Copia Project URL e anon public key.
5) Apri config.js e sostituisci:
   - INCOLLA_PROJECT_URL
   - INCOLLA_ANON_PUBLIC_KEY
6) Carica su GitHub questi file:
   - index.html
   - style.css
   - app.js
   - config.js
   - manifest.json
7) Apri il sito GitHub Pages dall'iPhone e aggiungilo alla schermata Home.

Nota: questa versione usa policy pubbliche per semplicità. Dopo il test conviene proteggerla con login o chiave privata.
